# Booking system prime prompt

```text
Build a minimal booking flow for a single-service business.

The business owner should be able to define available appointment slots, and a
customer should be able to reserve one available slot. The system should prevent
double booking. A customer should be able to cancel a booking before the
appointment starts.

Keep the first version small. Do not add payments, staff scheduling, package
services, waitlists, reminders, or recurring bookings unless they are necessary
to make the core booking flow correct.
```
